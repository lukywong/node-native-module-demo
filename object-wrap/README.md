# Introduction
How to return non-static C++ member to node on condition.

# How to run
Run this code under node version to 0.12.5

``` bash
$ nvm use 0.12.5
```

``` bash
$ npm run config
```

Build the project

``` bash
$ npm run build
```

Run the native moudle by nodejs

``` bash
$ npm run exe
```
